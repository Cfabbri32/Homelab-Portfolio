# Cybersecurity Honeypot with Wazuh SIEM

## Overview

This project focuses on building a cybersecurity monitoring environment using a Honeypot and Wazuh SIEM within my Proxmox homelab.

The goal was to gain hands-on experience with security monitoring, event collection, threat detection, and alerting by simulating malicious activity and observing how security tools detect and respond to it.

## Technologies Used

- Proxmox VE
- Wazuh
- Linux
- Virtual Machines
- Security Monitoring
- Network Analysis

## Skills Demonstrated

- SIEM Administration
- Security Monitoring
- Threat Detection
- Log Analysis
- Linux Administration
- Virtualization
- Cybersecurity Operations
