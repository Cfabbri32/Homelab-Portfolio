# Cybersecurity Honeypot with Wazuh SIEM

## Overview

This project focuses on building a cybersecurity monitoring environment using a dedicated Honeypot and Wazuh SIEM hosted within my Proxmox homelab.

The objective was to gain hands-on experience with threat detection, event monitoring, log analysis, compliance assessment, and security investigations by simulating real-world security monitoring workflows.

---

## Technologies Used

- Proxmox VE
- Wazuh SIEM
- Ubuntu Server 22.04
- Linux
- Threat Hunting
- File Integrity Monitoring (FIM)
- Security Configuration Assessment (SCA)
- CIS Benchmarks
